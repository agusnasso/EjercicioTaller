HTML

HTML (HyperText Markup Language o Lenguaje de Marcado de Hipertexto) es el código estándar que se utiliza para estructurar y desplegar el contenido de las páginas web. Piensa en él como el esqueleto o plano de una construcción que define dónde van los títulos, los párrafos, las imágenes y los enlaces.

CSS

CSS (siglas de Cascading Style Sheets o Hojas de Estilo en Cascada) es el lenguaje que se utiliza para definir el aspecto visual y el diseño de una página web.

javascript

JavaScript (JS) es un lenguaje de programación que se utiliza principalmente para crear interactividad en páginas web. Junto con HTML y CSS, forma el núcleo de casi todo lo que ves y haces en internet.
